# <PassOP/> - Your Personal Password Manager

<PassOP/> is a simple, secure, and client-side password manager designed to help you save and manage your website credentials directly in your browser. All your data is stored locally using `localStorage`, ensuring that your sensitive information never leaves your device.

## üöÄ Features

* **Save Credentials:** Easily save new website credentials (URL, username, password).
* **View & Manage:** View all your saved passwords in an organized and responsive table.
* **Copy to Clipboard:** Copy any website URL, username, or password to the clipboard with a single click.
* **Edit & Update:** Click the edit icon to repopulate the form with existing data for quick updates.
* **Secure Deletion:** Delete unwanted passwords with a confirmation step to prevent accidents.
* **Toggle Visibility:** Show or hide your password in the input form with the click of an eye icon.
* **Local Storage:** All data is stored securely in your browser's `localStorage`. No database or internet connection is required.
* **Toast Notifications:** Get user-friendly feedback for actions like saving, deleting, and copying.
* **Fully Responsive:** Designed to work smoothly on both desktop and mobile devices.
